📦 ReBoX

Donnez une seconde vie aux cartons.

1. Introduction

Ce document contient tous les éléments nécessaires pour présenter le projet ReBoX lors d'une soutenance ou d'une présentation publique.

🎯 Le Problème

  • Gaspillage : Les entreprises paient pour recycler des cartons encore utilisables.
  • Coût : Les particuliers paient cher des cartons neufs pour leurs besoins ponctuels.
  • Impact Éco : Le recyclage consomme de l'énergie ; le réemploi est bien plus vertueux.

💡 La Solution ReBoX

ReBoX est une marketplace écologique, économique et locale dédiée au marché de la seconde main du carton.

  • Le concept : Le "Vinted" du carton d'emballage.
  • L'ambition : Créer un circuit court entre les producteurs de surplus et les demandeurs.

2. Démonstration Fonctionnelle

🗺️ Recherche Géolocalisée

Visualisation sur une carte interactive (Leaflet) pour favoriser le circuit ultra-court et réduire l'empreinte carbone.

4. Préparation aux Questions

Pourquoi Supabase ?

Rapidité de mise en place. Auth, Storage et WebSockets intégrés pour se concentrer sur le métier.

Sécurité des paiements ?

Utilisation de Stripe Checkout. Aucune donnée bancaire sur nos serveurs.

Vraiment mobile ?

Oui, Capacitor permet de compiler le code web en application native APK/IPA.
